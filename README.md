# 🧠 Trabalho Final - Lógica de Programação | Serratec

Projeto desenvolvido como **trabalho final da disciplina de Lógica de Programação** da residência **Serratec**.

O objetivo do trabalho é implementar diferentes exercícios utilizando **Portugol**, explorando conceitos fundamentais de programação como:

- Estruturas de decisão
- Estruturas de repetição
- Vetores e matrizes
- Funções
- Recursividade
- Manipulação de texto
- Validação de dados

# 🛠 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando apenas:

- **Portugol**

Os algoritmos foram escritos seguindo a estrutura padrão da linguagem, contendo declaração de variáveis, leitura de dados, processamento e exibição de resultados.
